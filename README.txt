= Revision Games =

*It's simple - add words, practice with games, and learn!* After activation, each user will see individualized content: a **personalized dictionary**. Firstly, the user needs to create groups related to a specific subject. Then, he needs to add words with their translations within the group. Then, comes the fun part: Vocabulary quizzes!
There are two types of games available yet: **Find the Translation** and **Find the Word**. In the first case, you need to find the right translation amongst a list of four possible answers. While during the second vocabulary game, you need to find the right word amongst a list of four possible answers.

= Find the Translations Game =

Learning a new language is now even easier with the Personal Dictionary plugin. The users now can learn the words in a fun way. All they need to do is to find the translation of the new word in their own language.

= Calculation = 

Most probably you are asking yourself - *after how many attempts the word is considered as already learned*? It's all up to you.  You need to define the following inputs: **Learnt points, Right answer points, and Wrong answer points**. To give you an idea, let's assign 10 to the Learnt points, 1 to the Right, and 0.23 to the Wrong one. When the student answers a question 10 times correctly(10 x 1), then the given word will be considered already learned and will not appear in games anymore. Anyway, it will remain in the word bank, and the user can reset it anytime. This is the best way for *engaging students*, isn't it?

= Glossary VS Dictionary =

Let's find out the difference between glossaries and dictionaries. A glossary is a list of terms referring to a particular subject in alphabetical order. On the other hand, a dictionary is a word pool with its corresponding meanings, translation, or explanation; for instance, Oxford English Dictionary or Cambridge dictionary.

== Installation ==


There are two ways to install Personal Dictionary: the easy way, when you install Personal Dictionary from your WordPress dashboard, and the not so easy way, when you install it from WordPress.org.

* 1.1 The easiest way to enjoy Personal Dictionary:
* 1.1.1	Login to your WordPress dashboard
* 1.1.2	Go to Plugins
* 1.1.3	Add New
* 1.1.4	Search for Personal Dictionary
* 1.1.5	Click to install
* 1.2 The second way:
* 1.2.1Download the zip file from https://wordpress.org/plugins/personal-dictionary/
* 1.2.2 Go to Plugins
* 1.2.3 Add New
* 1.2.4 Upload plugin
* 1.2.5 Choose file ays personal_dictionary.zip
* 1.2.6	Click to install

* 1.3 In order to install the Personal Dictionary from Wordpress.org you must unzip the archive and copy the personal-dictionary folder into your plugins folder (\wp-content\plugins).
After one ofthis steps you can activate the plugin from the Plugins menu.
Once activated configure any options as desired and you can enjoy our Personal Dictionary.
